# 🌙 Luna — Cycle Calendar

A beautiful, private menstrual cycle tracker that works as an installable app on your phone. No accounts, no server — all data stays on your device.

## Features
- 📅 Monthly calendar with color-coded cycle phases
- 🩸 Log period dates, flow level, and notes per cycle
- 📈 Trend charts (cycle length, period duration, flow distribution)
- ✨ Phase tips — nutrition, movement & self-care per phase
- ⚠️ Irregularity warnings
- 🖨 Print monthly view / Export CSV / Backup JSON
- 📱 Installable as a phone app (PWA — no app store needed)
- 🔒 100% offline — data saved locally in your browser

## 🚀 Deploy to GitHub Pages (Free Hosting)

### Step 1 — Create a GitHub account
Go to [github.com](https://github.com) and sign up (free).

### Step 2 — Create a new repository
1. Click the **+** button → **New repository**
2. Name it: `luna-cycle` (or anything you like)
3. Set it to **Public**
4. Click **Create repository**

### Step 3 — Upload the files
1. On your new repo page, click **uploading an existing file**
2. Drag and drop these 4 files:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icons/icon-192.png`
   - `icons/icon-512.png`
3. Click **Commit changes**

### Step 4 — Enable GitHub Pages
1. Go to **Settings** → **Pages**
2. Under "Source", select **Deploy from branch**
3. Choose **main** branch → **/ (root)**
4. Click **Save**

### Step 5 — Get your URL
After ~1 minute, your app is live at:
```
https://YOUR-USERNAME.github.io/luna-cycle/
```

## 📱 Install on Your Phone

### Android (Chrome)
1. Open the GitHub Pages URL in Chrome
2. Tap the **three dots menu** → **Add to Home Screen**
3. Tap **Install** — Luna appears on your home screen!

### iPhone (Safari)
1. Open the URL in **Safari** (must be Safari, not Chrome)
2. Tap the **Share button** (box with arrow)
3. Scroll down → tap **Add to Home Screen**
4. Tap **Add** — done!

## 🔒 Privacy
All your cycle data is stored **only on your device** using localStorage.
Nothing is sent to any server. Ever.

Use **Export → Backup JSON** to save your data before clearing your browser.
